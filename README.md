# Combat Refactor

Combat Refactor is a RimWorld mod that reimagines the game's combat experience. It improves both combat depth and runtime performance.

## Features

### Combat resolution

**Hit model**

Shots are resolved through spread, cover interception, and target exposure.

**Armor model**

Armor uses deterministic resolution instead of random rolls.

### Weapon handling

**Fire selector**

Switch weapons between single, burst, and automatic fire.

**Magazine**

Weapons use magazines with explicit capacity and reload time.

### Performance

**Projectile runtime**

Projectile cover traversal and removal are optimized for lower overhead.
